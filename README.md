# Linux Administration Labs (RHCSA/RHCE/LFCS/LPIC/Linux+)

This repository contains hands-on Linux system administration labs based on enterprise scenarios.
The focus is on practical skills: user management, permissions, services, networking, storage,
automation and troubleshooting.

## Skills Covered
- Linux filesystem & permissions
- Users & groups administration
- systemd services & process management
- Networking basics (IP, DNS, firewall)
- Storage management (partitions, LVM, mounts)
- Bash scripting & automation (cron)

## Structure
- `modules/` contains module-based labs
- Each lab includes:
  - `README.md` (what/why/result)
  - `commands.sh` (commands used)
  - optional scripts and configs (sanitized)
